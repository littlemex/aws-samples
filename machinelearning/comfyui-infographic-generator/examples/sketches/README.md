# Sketch Examples

このディレクトリには、ControlNet Scribble用のスケッチを配置します。

## 📁 用途

- ControlNet Scribble用の入力画像
- ラフスケッチからの高品質化
- アイデアスケッチの具現化

## 🎨 スケッチの作り方

### 1. 手書きスケッチ

- 紙とペンでラフスケッチ
- スマホで撮影
- このディレクトリに配置

### 2. デジタルスケッチ

- iPad / ペンタブレットで作成
- PNG または JPG で保存
- 白背景、黒線推奨

### 3. 既存図のトレース

- PowerPoint などでラフなボックス配置
- スクリーンショット
- エッジがはっきりしている画像

## 💡 良いスケッチの例

✅ **推奨:**
- 明確な輪郭線
- シンプルな形状
- 白背景、黒またはグレーの線
- 1024x1024 以上の解像度

❌ **避けるべき:**
- ぼやけた線
- 複雑すぎる詳細
- 低解像度
- 色が多すぎる

## 📝 ワークフロー

1. **スケッチ作成**
   ```
   [アイデア] → [手書き/デジタルスケッチ] → [ファイル保存]
   ```

2. **ComfyUIで処理**
   ```
   [スケッチ読み込み] → [ControlNet Scribble] → [高品質化]
   ```

3. **微調整**
   ```
   [プロンプト調整] → [パラメータ調整] → [再生成]
   ```

## 🔧 推奨ツール

- **手書き:** 紙 + ペン + スマホカメラ
- **デジタル:** iPad + Apple Pencil, Wacom タブレット
- **PC:** GIMP, Krita, Photoshop
- **オンライン:** Excalidraw, tldraw

---

**ヒント:** シンプルなスケッチほど良い結果が得られます！
